Spacing

A .margin- or .padding- consists of it's side and size properties.

sizes

xs
set the margin or padding to $size-xl / 6
sm
set the margin or padding to $size-xl / 3
md
set the margin or padding to $size-xl / 2
lg
set the margin or padding to $size-xl / 1.5
xl
set the margin or padding to $size-xl

sides

top
for classes that set margin-top or padding-top
right
for classes that set margin-right or padding-right
bottom
for classes that set margin-bottom or padding-bottom
left
for classes that set margin-left or padding-left
x
for classes that set both *-left and *-right
y
for classes that set both *-top and *-bottom

global

margin-{size}, padding-{size}
set a margin or padding on all 4 sides of the element
